# rex-bot

First create config.js 

```js
export default {
  token: "your token here (not bot token)",
  prefix: "!",
  allowedUsers: ["Your user id here"]
};
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```

This project was created using `bun init` in bun v1.3.1.
